Master Thesis

IMPROVING SEN2CUBE.AT WEB APPLICATION VISUALIZATION CAPABILITIES

Data cube is the new approach of organizing and managing multidimensional data, in Earth Observation (EO) domain, satellite imagery that stored in server on each pixel is associated with coordinate. EO data cube propose a possibility to tackle big data challenge by translates the coordinate position from digital globe to the internal file structure, collects the images, and conveniently present them as continuous map (semantixcube.net). Furthermore, EO Data Cube also possible to do such as processing satellite imagery into semantic enrichment information on the fly. By combining EO data cube and website technology, a semantic enrichment satellite imagery could be produced and visualized as web map directly in web application system.

Sen2Cube.at is the first remote sensing web application designed as general-purpose semantic Earth Observation data cube on the fly particularly for Sentinel 2 imagery. Users do not need technical programming skill to collect this semantic information. By providing Graphic User Interface (GUI), user-define image query could be obtained through the website and retrieve spatio-temporal information directly in result interface as image map and possibly to download as raster data (GeoTIFF), WMS link, or QGIS project. Along with the raising number of active users, improving the image map visualization become an emerge step toward enhancing the web application user experience. Instead continue in any spatial desktop-software to refining the image map result, to do it in same environment as the analysis process goes will be a plus value of Sen2Cube.at.

Cartography takes important role in due to improve user experience by combining with web map design. Understanding the user needs is the important value regarding to create interactive and efficient web map application. Acknowledge the user’s characteristics and background which affect their ability to perceive and/or to comprehend the geographical information inherent in the map is not only relevant for the design and development of cartographic tools for the web (Kaark, 2001). Through this study, we try to examine the user’s need in due to improve user experience by performing user evaluation followed by developing visualization and cartographic tools both as concept and prototype. User evaluation including SWOT analysis and user questionnaire for determining the threshold of web map visualization. Both results will be input in order to generate concept and creating the visualization and cartographic tools prototype as the suggestion for Sen2Cube.at team in due to improving visualization capabilities.
